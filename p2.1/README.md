## ¿Qué es esya app?

Es una aplicació nque muestra como titulo la parte solicitada de "Mi Primer Hola Mundo" y al guardar y hacer el reload se actualiza de manera automatica en timepo real a los nuevos cambio aplicacdos en este caso colocando r en la consola donde se esta ejecutando asi cambiando "Mi primer Aplicación"

## Como se ejecuta

Para ejecutarla solamente hay que abrirla que se encunta en la carpeta llamada Practica_Hello_Word\climate_app de este repositorio una vez estando ahi la ejecutamos desde la consola con el comando flutter run y nos aseguramos que elemulador este abierto en android estudio o nuestro dispositivo este conectado mediante un cable USB y este activada la depuración desde opciónes de administrador
